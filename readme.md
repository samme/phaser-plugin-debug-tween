Phaser Debug Tween Plugin
=========================

```javascript
debug.tween(tween, x, y)

debug.tweenData(tweenData, x, y)

debug.tweenDataInfo(tweenData, x, y)

debug.tweenDataValues(tweenData, x, y)

debug.tweenInfo(tween, x, y)
```
